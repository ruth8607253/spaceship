## 遊玩方法

- 按左右鍵操控小飛船移動，按上可發射子彈打隕石，有時候會掉出增加子彈數(閃電)與恢復血量(盾牌)的道具。

- 打越多隕石分數越高，隕石越大科分數越高。

- 被撞到直到寫條耗盡3次時遊戲結束。
